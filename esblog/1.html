Visual Studio快捷键汇总

发表于2025-05-15|更新于2025-05-15
|字数总计:503|阅读时长:1分钟|阅读量:

以下是Visual Studio常用快捷键的汇总，涵盖编辑、调试、文件操作、导航和视图等多个方面，帮助提升开发效率：

常用编辑快捷键

  • 注释代码：Ctrl + K, Ctrl + C
  • 取消注释：Ctrl + K, Ctrl + U
  • 代码格式化：
    • 格式化整个文档：Ctrl + K, Ctrl + D
    • 格式化选中内容：Ctrl + K, Ctrl + F
  • 复制整行代码（无需选中）：将光标放在行上，Ctrl + C
  • 剪切整行代码：将光标放在行上，Ctrl + X
  • 删除整行代码：Ctrl + Shift + K
  • 撤销：Ctrl + Z
  • 重做：Ctrl + Y 或 Ctrl + Shift + Z
  • 查找：Ctrl + F
  • 查找全部文件：Ctrl + Shift + F
  • 替换：Ctrl + H
  • 转到定义：F12
  • 转到声明：Ctrl + F12
  • 查找所有引用：Shift + F12
  • 跳转到指定行：Ctrl + G
  • 插入代码片段：Ctrl + K, Ctrl + X
  • 切换书签：Ctrl + K, Ctrl + K
  • 切换行注释（VS2022及以上）：Ctrl + / 或 Ctrl + K, Ctrl + /

调试快捷键

  • 开始调试：F5
  • 开始执行但不调试：Ctrl + F5
  • 单步执行（进入函数）：F11
  • 单步跳出（跳出函数）：Shift + F11
  • 逐过程执行：F10
  • 停止调试：Shift + F5
  • 切换断点：F9
  • 删除所有断点：Ctrl + Shift + F9
  • 运行到光标处：Ctrl + F10
  • 设置下一语句：Ctrl + Shift + F10
  • 重启调试：Ctrl + Shift + F5

文件和项目操作快捷键

  • 新建项目：Ctrl + Shift + N
  • 新建文件：Ctrl + N
  • 打开文件：Ctrl + O
  • 打开项目：Ctrl + Shift + O
  • 保存当前文件：Ctrl + S
  • 全部保存：Ctrl + Shift + S
  • 添加新项：Ctrl + Shift + A
  • 添加现有项：Shift + Alt + A

视图和窗口快捷键

  • 打开解决方案资源管理器：Ctrl + Alt + L
  • 打开属性窗口：F4
  • 切换活动窗口：Ctrl + Tab
  • 打开错误列表：Ctrl + W, E
  • 打开输出窗口：Ctrl + W, O
  • 打开任务列表：Ctrl + W, T
  • 打开工具箱：Ctrl + W, X
  • 打开断点窗口：Ctrl + D, B
  • 打开即时窗口：Ctrl + D, I

代码修复和导航

  • 快速修复/智能提示：Ctrl + . 或 Ctrl + +
  • 查找下一个结果：F3
  • 查找上一个结果：Shift + F3
